# Higienize SC

Professional website project developed for an upholstery cleaning company based in Florianópolis, Brazil.

This project is being built using ASP.NET Core MVC with a modern responsive interface focused on performance, visual presentation, and mobile experience.

## Main Features

* Responsive and Mobile First layout
* Professional services presentation
* Dynamic testimonials section
* Customer comment system
* FAQ section
* Animated service cards
* Responsive mobile navigation menu
* Online deployment for testing and preview

## Technologies Used

* ASP.NET Core MVC
* C#
* HTML5
* CSS3
* Razor Pages
* Bootstrap Icons
* Docker
* Render Hosting

## Project Status

Project currently under development.

Future improvements planned:

* Admin panel
* Google authentication
* Comment approval system
* Dynamic gallery management
* Database integration
* Custom domain configuration

O projeto está sendo desenvolvido utilizando ASP.NET Core MVC com uma interface moderna, responsiva e focada em desempenho, apresentação visual e experiência mobile.
